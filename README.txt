Title: Scheduling Application

Purpose: Used to schedule appointments associated with associated customers.
Contains the ability to add/modify/delete appointments and appointments as well contains additional reports.

Author: Kenneth Lovelace

Contact Information: klov191@wgu.edu

Application Version 1.0.0

Date: 3/22/2023

IDE/Version: IntelliJ Community 2021.1.3

JDK Version: jdk-11.0.11

JavaFX Version: JavaFX-SDK-11.0.2

How to run program: Launch application, login with username = "test" password = "test"
You will be navigated to the appointments screen displaying all appointments.
From here you can view add/modify/delete appointments or navigate to the customers screen
where all of those options are also present. From the appointments screen you can also view the generated reports.

Description of Additional Report (View number of customers by country): Allows the user to view the number of customers by country selected

MySql driver version: mysql-connector-java-8.0.25
